# Flink Connector for MaxCompute

The Flink Connector enables streaming data between Apache Flink and Alibaba Cloud MaxCompute.

## Features

- Read MaxCompute tables as Flink DataStreams
- Write Flink DataStreams to MaxCompute tables
- Support for batch and streaming modes

## Quick Start

### Maven Dependency

```xml
<dependency>
    <groupId>com.aliyun.odps</groupId>
    <artifactId>flink-connector-odps</artifactId>
    <version>2.0.5</version>
</dependency>
```

## Requirements

- Apache Flink 1.x
- JDK 1.8 or later
- Alibaba Cloud MaxCompute account
