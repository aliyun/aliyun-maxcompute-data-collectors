# Flume Plugin for MaxCompute

:::caution Archived
This connector is no longer actively maintained. The source code remains available for reference.
:::

The Flume Plugin enables collecting and ingesting log data into Alibaba Cloud MaxCompute using Apache Flume.

## Features

- MaxCompute Sink for Apache Flume
- Batch upload of log data
- Configurable batch sizes and intervals

## Requirements

- Apache Flume 1.x
- JDK 1.6 or later
- Alibaba Cloud MaxCompute account
