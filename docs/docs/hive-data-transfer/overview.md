# Hive Data Transfer for MaxCompute

:::caution Archived
This connector is no longer actively maintained. The source code remains available for reference.
:::

The Hive Data Transfer module provides UDTFs (User-Defined Table-Generating Functions) for transferring data between Apache Hive and Alibaba Cloud MaxCompute.

## Features

- Transfer data from Hive to MaxCompute
- Transfer data from MaxCompute to Hive
- UDTF-based approach for seamless integration

## Requirements

- Apache Hive
- JDK 1.6 or later
- Alibaba Cloud MaxCompute account
