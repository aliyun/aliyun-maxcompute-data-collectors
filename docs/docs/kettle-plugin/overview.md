# Kettle Plugin for MaxCompute

:::caution Archived
This connector is no longer actively maintained. The source code remains available for reference.
:::

The Kettle Plugin provides ETL integration between Pentaho Data Integration (Kettle/PDI) and Alibaba Cloud MaxCompute.

## Features

- MaxCompute Input/Output steps for Kettle
- Visual ETL pipeline design with MaxCompute support
- Drag-and-drop data flow configuration

## Requirements

- Pentaho Data Integration (Kettle) 7.x or later
- JDK 1.8 or later
- Alibaba Cloud MaxCompute account
