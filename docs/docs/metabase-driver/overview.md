# Metabase Driver for MaxCompute

The Metabase Driver allows connecting Metabase to Alibaba Cloud MaxCompute for business intelligence and data analytics.

## Features

- Native Metabase database driver for MaxCompute
- SQL query support through Metabase UI
- Dashboard and visualization capabilities

## Requirements

- Metabase 0.44 or later
- Alibaba Cloud MaxCompute account
