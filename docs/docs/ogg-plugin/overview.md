# OGG Plugin for MaxCompute

:::caution Archived
This connector is no longer actively maintained. The source code remains available for reference.
:::

The OGG Plugin enables real-time data replication from Oracle to Alibaba Cloud MaxCompute via Oracle GoldenGate.

## Features

- Real-time CDC (Change Data Capture) from Oracle
- Automatic table creation in MaxCompute
- Support for INSERT, UPDATE, and DELETE operations

## Requirements

- Oracle GoldenGate 12.x or later
- Oracle Database
- JDK 1.8 or later
- Alibaba Cloud MaxCompute account
