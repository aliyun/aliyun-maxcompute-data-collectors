# Presto Connector for MaxCompute

The Presto Connector allows querying Alibaba Cloud MaxCompute data using PrestoDB.

## Features

- Execute SQL queries against MaxCompute tables via PrestoDB
- Support for predicate pushdown
- Type mapping between Presto and MaxCompute

## Requirements

- PrestoDB
- JDK 1.8 or later
- Alibaba Cloud MaxCompute account
