# Sqoop for MaxCompute

:::caution Archived
This connector is no longer actively maintained. The source code remains available for reference.
:::

The Sqoop plugin enables bulk data transfer between Alibaba Cloud MaxCompute and relational databases (MySQL, Oracle, SQL Server, etc.).

## Features

- Import data from RDBMS to MaxCompute
- Export data from MaxCompute to RDBMS
- Support for incremental imports

## Requirements

- Apache Sqoop 1.x
- JDK 1.6 or later
- Alibaba Cloud MaxCompute account
