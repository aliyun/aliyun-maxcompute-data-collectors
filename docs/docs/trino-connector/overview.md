# Trino Connector for MaxCompute

The Trino Connector allows querying Alibaba Cloud MaxCompute data using Trino (formerly PrestoSQL).

## Features

- Execute SQL queries against MaxCompute tables via Trino
- Support for predicate pushdown
- Type mapping between Trino and MaxCompute

## Requirements

- Trino 3xx or later
- JDK 11 or later
- Alibaba Cloud MaxCompute account
