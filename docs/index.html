MaxCompute Connector for Google Sheets MaxCompute Google Sheets 连接器

Query Alibaba Cloud MaxCompute data directly in Google Sheets. Built by Alibaba Cloud. 在 Google Sheets 中直接查询阿里云 MaxCompute 数据。由阿里云开发。

Install from Marketplace 从应用商店安装

Key Features 核心功能

SQL Query Execution SQL 查询执行

Execute SQL queries in the sidebar and results are automatically written to your sheet (up to 10,000 rows). 在侧边栏执行 SQL 查询，结果自动写入表格（最多 10,000 行）。

Data Catalog Browser 数据目录浏览

Navigate through Project → Schema → Tables with a three-level tree structure. 通过三层树形结构浏览 Project → Schema → Tables。

One-Click SQL Generation 一键生成 SQL

Click on a table name to automatically generate a SELECT statement. 点击表名自动生成 SELECT 查询语句。

Quick Field Insert 字段快速插入

Click on any field name to insert it directly into the SQL editor. 点击字段名直接插入到 SQL 编辑器中。

Partition Table Support 分区表支持

View partition columns and browse partition lists for partitioned tables. 查看分区列，浏览分区表的分区列表。

Logview Integration Logview 集成

Instance ID is clickable and redirects to MaxCompute console for detailed job information. Instance ID 可点击跳转到 MaxCompute 控制台查看作业详情。
